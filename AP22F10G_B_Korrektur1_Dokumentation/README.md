# AP22F10G-B Korrektur 1 – Dokumentation

Release: `NK-Pro V99.4.59`

Technische Ausgangsbasis: `NK-Pro_V99_4_58_AP22F10G_B_Individuelle_Werte.zip`

Realer Datenerhaltungs- und Browser-Testbestand:
`nk-pro-gesamtbestand-2025-V99.4.58-2026-07-19-16-16-12.json`

Die Korrektur umfasst ausschließlich die Seite „Individuelle Werte“:

- kompaktere Gesamtbreite nach NK-Pro-Standard,
- verlässlicher linker und rechter Tabelleninnenabstand,
- blauer Punkt für Kaltwasserzähler,
- roter Punkt für Warmwasserzähler,
- ausgegraute und gesperrte übernommene Anfangsstände,
- bewusste Freigabe einer Überschreibung über die Sonderfallaktion.

Das Datenschema bleibt Version 5. Beim Laden oder Importieren werden keine bestehenden Nutzerdaten verändert.
