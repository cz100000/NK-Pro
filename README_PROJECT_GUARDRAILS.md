# NK-Pro Projektleitplanken

Dieses Paket ergänzt die technische Projektführung vor V95.

## Enthalten

- `ARCHITECTURE.md` – verbindliche Architekturgrundsätze
- `ROADMAP.md` – schrittweiser Entkopplungs- und Vereinfachungsplan
- `RELEASE_CHECKLIST.md` – verpflichtender Architektur- und Release-Check
- `CHANGELOG_PROJECT_GUARDRAILS.md` – Änderungsnotiz

## Einspielen

Die Dateien in den lokalen Repository-Ordner `NK-Pro` kopieren.

Danach in GitHub Desktop:

- Summary: `NK-Pro Projektleitplanken vor V95`
- `Commit to main`
- `Push origin`

Die bestehende `index.html` und die V94-App werden dadurch nicht verändert.
