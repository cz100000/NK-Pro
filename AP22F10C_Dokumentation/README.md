# AP22F10C – Korrektur Leerstandserfassung

Release V99.4.47 korrigiert die Seite „Individuelle Werte“: Vollständig leer stehende Wohnungen ohne Mietverhältnis werden als eigene Abrechnungsfälle angezeigt. Externe Heiz- und Warmwasserkosten sowie Wasserwerte können der Wohnung und dem Leerstandszeitraum zugeordnet werden, ohne künstlichen Mieterdatensatz.

Datenschema 5 bleibt unverändert.
