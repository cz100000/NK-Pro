# NK-Pro UI-Komponentenkatalog

| Komponente | Basisklasse | Varianten | Zweck | AP22A-Status |
|---|---|---|---|---|
| Button | `nk-ui-button` | primary, secondary, danger, ghost, icon | Aktionen | Fundament |
| Formularfeld | `nk-ui-field` | default, invalid, disabled, readonly | Beschriftete Eingaben | Fundament |
| Karte | `nk-ui-card` | default, muted, interactive | Inhaltliche Gruppierung | Fundament |
| Accordion | `nk-ui-accordion` | default | Klappbarer Bereich | Fundament |
| Tabelle | `nk-ui-table` | default, compact | Tabellarische Daten | Fundament |
| Status | `nk-ui-status` | neutral, info, success, warning, danger | Kompakter Zustand | Fundament |
| Hinweis | `nk-ui-notice` | info, success, warning, danger | Kontextmeldung | Fundament |
| Toolbar | `nk-ui-toolbar` | default, compact | Filter und Aktionen | Fundament |
| Dialog | `nk-ui-dialog` | default, danger | Modale Interaktion | Fundament |
| Leerzustand | `nk-ui-empty-state` | default | Fehlende Inhalte | Fundament |

AP22A definiert die Schnittstellen. Produktive Seitenmigrationen folgen ab AP22B.
